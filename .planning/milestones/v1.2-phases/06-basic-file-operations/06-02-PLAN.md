        *tview.Box

    ## NewInputDialog

    ```go
    func NewInputDialog(app *tview.Application) *InputDialog {
        id := &InputDialog{
            Box:     tview.NewBox(),
            app:     app,
            visible: false,
        }
        id.inputField = tview.NewInputField()
        id.inputField.SetFieldBackgroundColor(tcell.Color236)
        id.inputField.SetFieldTextColor(tcell.Color255)
        id.inputField.SetDoneFunc(func(key tcell.Key) {
            if key == tcell.KeyEnter {
                text := id.inputField.GetText()
                if text != "" && id.onSubmit != nil {
                    id.onSubmit(text)
                }
                id.Hide()
            } else if key == tcell.KeyEscape {
                if id.onCancel != nil {
                    id.onCancel()
                }
                id.Hide()
            }
        })
        id.SetBorder(true).
            SetBorderColor(tcell.Color238).
            SetTitleColor(tcell.Color250).
            SetBackgroundColor(tcell.Color232)
        return id
    }
    ```

    ## Draw 渲染

    居中 popup，宽度 60%（max 60），高度 7（固定）。

    布局（参考 TransferModal conflictDialog 的三行布局）：
    - Row 0: title（如 "Rename" 或 "New Directory"）用 `tcell.Color255`
    - Row 1: label + InputField（label 如 "Name: "）
    - Row 2: 空行
    - Row 3: "[Enter] Confirm  [Esc] Cancel" 用 `tcell.Color245`

    InputField 渲染：使用 `tview.Print` 渲染 label，然后手动定位 InputField 的 Draw 到 label 右侧。由于 InputField 是 tview.Primitive，可以直接调用 `id.inputField.SetRect(ix+labelWidth, iy+1, iw-labelWidth-2, 1)` 然后 `id.inputField.Draw(screen)`。

    **关键：InputField 不使用 tview focus 系统**。通过 HandleKey 手动调用 `inputField.InputHandler()` 路由按键（per Claude's Discretion + ARCHITECTURE.md Pattern 2）。InputField 无需调用 `SetFocus()`。

    ## HandleKey（per Pitfall 3: Enter/Esc 双重触发）

    ```go
    func (id *InputDialog) HandleKey(event *tcell.EventKey) *tcell.EventKey {
        if !id.visible { return event }
        id.inputField.InputHandler(event, func(tview.Primitive) {})
        return nil // consume all keys
    }
    ```

    **关键设计：** Enter/Esc 只通过 InputField 的 `doneFunc` 处理（在 NewInputDialog 中设置），HandleKey 不重复检查。这避免了 Pitfall 3（双重触发）。
