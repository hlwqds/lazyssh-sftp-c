**handleGlobalKeys** (DRB-03, D-10):
```go
func (drb *DualRemoteFileBrowser) handleGlobalKeys(event *tcell.EventKey) *tcell.EventKey {
    // Overlay priority (same pattern as FileBrowser)
    if drb.inputDialog != nil && drb.inputDialog.IsVisible() {
        return drb.inputDialog.HandleKey(event)
    }
    if drb.confirmDialog != nil && drb.confirmDialog.IsVisible() {
        return drb.confirmDialog.HandleKey(event)
    }

    switch event.Key() {
    case tcell.KeyTab:
        drb.switchFocus()
        return nil
    case tcell.KeyESC:
        drb.close()
        return nil
    }
    switch event.Rune() {
    case 'd':
        drb.handleDelete()
        return nil
    case 'R':
        drb.handleRename()
        return nil
    case 'm':
        drb.handleMkdir()
        return nil
    case 's':
        drb.cycleSortField()
        return nil
    case 'S':
        drb.reverseSort()
        return nil
    }
    return event // pass to focused pane's InputCapture
}
```

func (drb *DualRemoteFileBrowser) switchFocus() {
    if drb.activePane == 0 {
        drb.sourcePane.SetFocused(false)
        drb.targetPane.SetFocused(true)
        drb.activePane = 1
        drb.app.SetFocus(drb.targetPane)

}

**handleDelete** (D-05):
Replicate FileBrowser.handleDelete pattern but simplified for remote-only:
1. Get `sftp := drb.currentSFTPService()`, `pane := drb.currentPane()`
2. Check `pane.IsConnected()` -- if not, call `drb.showStatusError("Not connected")` and return
3. Get selection: `row, _ := pane.GetSelection()`, `cell := pane.GetCell(row, 0)`
4. Extract `domain.FileInfo` from cell reference
5. Build full path: `joinPath(pane.GetCurrentPath(), fi.Name)` (use the existing `joinPath` function from remote_pane.go)
6. Build confirmation message with file type, size, modified time (same format as FileBrowser)
7. Wire `confirmDialog.SetOnConfirm` with goroutine calling `sftp.Remove(path)` or `sftp.RemoveAll(path)`
8. On success: call `drb.refreshAndReposition(drb.activePane, row)`
9. On error: call `drb.showStatusError(fmt.Sprintf("Delete failed: %s", trimError(err.Error(), 50)))`
10. Show confirm dialog: `drb.confirmDialog.Show("Delete?", message, detail)`

Also implement **handleBatchDelete** for Space-multi-selected files (same pattern as FileBrowser.handleBatchDelete). Use `pane.SelectedFiles()` to get multi-selected files.
