    **File 3: `file_browser_handlers.go`** -- Keyboard event handlers:

    Method `handleGlobalKeys(event *tcell.EventKey) *tcell.EventKey`:
    ```go
    switch event.Key() {
    case tcell.KeyTab:
        fb.switchFocus()
        return nil
    case tcell.KeyESC:
        fb.close()
        return nil
    }
    switch event.Rune() {
    case 's':
        fb.cycleSortField()
        return nil
    case 'S':
        fb.reverseSort()
        return nil
    }
    return event  // pass to focused pane's InputCapture
    ```

    Method `switchFocus()`:
    - If `activePane == 0`: set local unfocused, remote focused, `activePane = 1`, `app.SetFocus(remotePane)`

    - Call `sftpService.Close()` in goroutine (cleanup SSH process per Pitfall 3)
    - Call `onClose()` callback

    Method `cycleSortField()`:
    - Get active pane's sortMode, call `ToggleField()`, refresh that pane

    Method `reverseSort()`:
    - Get active pane's sortMode, call `Reverse()`, refresh that pane

    Method `updateStatusBarConnection(msg string)`:
