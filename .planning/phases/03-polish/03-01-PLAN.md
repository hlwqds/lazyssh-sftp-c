<objective>
为 TransferService 添加 context.Context 取消传播机制，实现传输中断、部分文件清理和双 Esc 取消确认 UI。

**为什么：** 当前 Esc 只是关闭 modal 遮罩层，goroutine 继续运行。真正的取消需要 context.Context 传播信号到传输循环，以及 UI 层的取消确认对话框防止误操作。

**输出：** TransferService 所有方法接受 ctx 参数，copyWithProgress 支持 ctx.Done() 中断，TransferModal 支持多模式切换（progress/cancelConfirm/canceled summary），FileBrowser 传递 cancel 函数到 UI 层。
</objective>

    **设计原理：** context.Context 是 Go 标准的取消传播模式。通过在 TransferService 所有方法签名中添加 `ctx context.Context` 作为第一个参数，传输 goroutine 可以通过 `ctx.Done()` channel 接收取消信号。`copyWithProgress` 的 32KB buffer 循环在每次 chunk 之间检查 `ctx.Done()`，实现最多 32KB 延迟的中断。取消后返回 `context.Canceled` 错误，调用方负责清理部分文件。

    **实现步骤：**

    1. **修改 `ports/transfer.go` 接口签名** -- 所有 4 个方法添加 `ctx context.Context` 第一个参数：
       ```go
       UploadFile(ctx context.Context, localPath, remotePath string, onProgress func(domain.TransferProgress)) error
       DownloadFile(ctx context.Context, remotePath, localPath string, onProgress func(domain.TransferProgress)) error
       UploadDir(ctx context.Context, localPath, remotePath string, onProgress func(domain.TransferProgress)) ([]string, error)
       DownloadDir(ctx context.Context, remotePath, localPath string, onProgress func(domain.TransferProgress)) ([]string, error)
       ```
       添加 `"context"` import。

    2. **修改 `transfer_service.go` 中 `copyWithProgress` 签名和实现**：
       - 添加 `ctx context.Context` 第一个参数
       - 在 for 循环开头（每次 Read 之前）添加取消检查：
         ```go
         select {
         case <-ctx.Done():
             return context.Canceled
         default:
         }
         ```
       - 保留现有 Done 事件发送（不发送 Done 事件当取消时）

    3. **修改 `UploadFile` 添加 ctx 传递和取消清理**（D-04, D-05）：
       - 方法签名添加 `ctx context.Context` 第一个参数
       - 调用 `copyWithProgress` 传递 ctx
       - copyWithProgress 返回后，**先关闭 remoteFile**（不使用 defer，避免竞态），再检查错误：
         ```go

         remoteFile.Close()

             ts.log.Infow("transfer canceled, cleaning up partial file", "path", remotePath)
             if removeErr := ts.sftp.Remove(remotePath); removeErr != nil {
         ```

       ```go
       func (tm *TransferModal) HandleKey(event *tcell.EventKey) *tcell.EventKey {
           if !tm.visible {
               return event
           }
           switch tm.mode {
           case modeCancelConfirm:
               switch event.Key() {
               case tcell.KeyEscape:
                   tm.ResumeProgress()
                   return nil
               case tcell.KeyEnter:
                   tm.cancelConfirmed = true
                   tm.onDismiss() // 触发取消流程
                   return nil
               }
               switch event.Rune() {
               case 'y':
                   tm.cancelConfirmed = true
                   tm.onDismiss()
                   return nil
               case 'n':
                   tm.ResumeProgress()
                   return nil
               }
               return nil // 消费所有其他按键

               tm.Hide()
               return nil
           case modeProgress:
               switch event.Key() {
               case tcell.KeyEscape:
                   tm.ShowCancelConfirm()
                   return nil
               }
               return event
           }
           return event
       }
       ```
