# Phase 1 -- UI Design Contract

> Visual and interaction contract for the dual-pane file browser TUI. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Note:** This is a terminal UI (TUI) application built on tview/tcell. Standard web UI concepts (px spacing, CSS typography) are expressed as terminal equivalents: tcell 256-color codes, character-based spacing, and tview component styling.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (TUI -- tview/tcell, not web) |
| Preset | not applicable |
| Component library | tview (Table, Flex, Box, TextView, Modal) |
| Icon library | Unicode emoji/symbols (existing pattern: cellPad + runewidth) |
| Font | Terminal monospace font (user-configured, not controlled by app) |

**Rationale:** Project is a Go TUI application (CLAUDE.md: "TUI (Terminal User Interface) -- Built with tview and tcell"). No web design system applies. The existing `initializeTheme()` in `tui.go` defines the color palette.

### Color Distribution

| Layer | Coverage | Colors |
|-------|----------|--------|
| Dominant (60%) | Background, borders, text | Color232, Color238, Color252 |
| Secondary (30%) | Status bar, header row, selected row | Color235, Color24, Color255 |
| Accent (10%) | Directories, selection markers, error/success | Color33, #FFD700, #FF6B6B, #A0FFA0 |

Accent reserved for:
- Directory names in file list (Color33 blue)
- Multi-select row markers (#FFD700 gold, via `*` prefix or bracket marker)
- Error messages (#FF6B6B red, via status bar)
- Success messages (#A0FFA0 green, via status bar)
- File browser pane borders when focused (brighter border, e.g. Color248)

## Layout & Spacing

### Layout Structure

```
+====================================================================+
| File Browser (full screen, replaces main view)                       |
|                                                                     |
| +========================+========================+                 |
| | /home/user             | user@host:~            |  <- pane titles  |
| +------------------------+------------------------+                 |
| | Name        Size  Date | Name        Size  Date |                 |
| | Permissions             | Permissions             |                 |
| +------------------------+------------------------+                 |
| | Documents/   -    12:00| Documents/   -    12:00|                 |
| | Pictures/    -    11:30| Pictures/    -    11:30|                 |
| | file.txt    1.2K 10:00| file.txt    1.2K 10:00|                 |
| | ...          ...  ...  | ...          ...  ...  |                 |
| +------------------------+------------------------+                 |
|                                                                     |
| [Tab] Switch pane  [h] Up  [.] Hidden  [s] Sort  [Esc] Back        |  <- status bar
+====================================================================+
```

Source: D-04 (50:50 Flex), D-05 (path as title), D-06 (full screen overlay)

### Spacing (character-based, multiples of 1 since terminal cells are atomic)

| Token | Value | Usage |
|-------|-------|-------|
| Cell gap | 1 char | Minimum gap between table columns (tview Table default) |
| Column separator | `"  "` (2 spaces) | Visual gap between Name/Size/Date/Permissions columns |
| Pane border padding | 1 char (tview default) | Padding inside pane borders |
| Status bar height | 1 row | Fixed bottom status bar (same as existing `status_bar.go`) |
| Header row height | 1 row | Fixed table header (via `SetFixed(1, 0)`) |

Exceptions: none -- terminal UI uses fixed cell dimensions.

## Typography

Terminal UI typography is determined by the user's terminal font and size. The application controls **style attributes** (bold, underline, color), not font metrics.

| Role | Style | Color | Usage |
|------|-------|-------|-------|
| Body (file names) | Normal | Color252 (PrimaryText) | File name column content |
| Label (metadata) | Normal | Color245 (SecondaryText) | Size, Modified date, Permissions columns |
| Heading (pane title) | Normal | Color250 (Title) | Current path in pane title bar |
| Heading (table header) | **Bold** | Color255 on Color235 BG | Column headers: Name, Size, Modified, Permissions |
| Selected row text | Normal | Color255 (on Color24 BG) | File name on active/selected row |
| Directory name | Normal | Color33 (blue) | Directory entries in Name column |
| Status bar | Normal | White `[white]` | Keybinding hints in status bar |
| Error text | Normal | #FF6B6B | Error messages in status bar / pane |
| Connecting text | Normal | Color245 | "Connecting..." placeholder in right pane |
| Selected file marker | Normal | #FFD700 | `*` prefix on multi-selected file rows |

Line heights: not applicable (terminal controls row height).

## Table Column Specification

Source: D-01 (tview.Table), D-02 (4 columns)

| Column | Header | Alignment | Width | Content Format |
|--------|--------|-----------|-------|----------------|
| Name | `"Name"` | Left | Expansion (auto-fill) | `{dir_name}/` or `{file_name}`, with `* ` prefix for selected |
| Size | `"Size"` | Right | ~6 chars | Human readable: `1.2K`, `3.4M`, `1.1G`, `0B` (dirs show `-`) |
| Modified | `"Modified"` | Left | ~16 chars | `2006-01-02 15:04` format |
| Permissions | `"Permissions"` | Left | ~10 chars | `drwxr-xr-x`, `-rw-r--r--` (os.FileMode.String()) |

Source: RESEARCH.md "Pattern 4: File List Filling", Claude's Discretion (human readable format)

**Directory sorting rule:** Directories always listed before files, then sorted by the active sort field within each group. Source: RESEARCH.md Pitfall 6.

**Hidden files:** Default hidden (filtered by `strings.HasPrefix(name, ".")`). Toggle with `.` key. Source: BROW-04, RESEARCH.md Open Question 3 (recommendation: `.` key).

## Keyboard Interaction Map

Source: D-11, D-12, D-13, D-14, D-15

| Key | Context | Action | Source |
|-----|---------|--------|--------|
| `F` (Shift+f) | Server list | Open file browser for selected server | D-14, UI-01 |
| `Esc` | File browser | Close browser, return to main view | D-15, D-06 |
| `Tab` | File browser (global) | Switch focus between left and right pane | D-12, UI-05 |
| `j` / `Down` | Focused pane | Move selection down (tview.Table built-in) | UI-03 |
| `k` / `Up` | Focused pane | Move selection up (tview.Table built-in) | UI-03 |
| `h` | Focused pane | Navigate to parent directory | D-11, BROW-03 |
| `Backspace` | Focused pane | Navigate to parent directory (same as `h`) | D-11, BROW-03 |
| `Enter` | Focused pane | Enter directory (Phase 1); transfer file (Phase 2) | BROW-01 |
| `Space` | Focused pane | Toggle multi-select on current file | D-13, UI-04 |
| `.` | Focused pane | Toggle hidden file visibility | BROW-04, RESEARCH rec |
| `s` | File browser (global) | Cycle sort field: Name -> Size -> Date -> Name | BROW-06 |
| `S` (Shift+s) | File browser (global) | Reverse sort direction | BROW-06 |
| `g` / `Home` | Focused pane | Jump to first row (tview.Table built-in) | tview default |
| `G` / `End` | Focused pane | Jump to last row (tview.Table built-in) | tview default |
| `PgUp` / `PgDn` | Focused pane | Scroll by page (tview.Table built-in) | tview default |

**Event propagation chain:**
1. `FileBrowser.SetInputCapture` -> handles Tab, Esc, s, S
2. `FocusedPane.SetInputCapture` -> handles h, Backspace, Space, `.`
3. `Table.InputHandler` -> handles j/k/arrow/Enter/PgUp/PgDn (built-in)

Source: RESEARCH.md Pattern 2, Pitfall 5

## Pane Title Format

Source: D-05 (path as pane Title)

| Pane | Title Format | Example |
|------|-------------|---------|
| Local (left) | `" {path} -- Sort: {field} [{dir}] "` | `" /home/user -- Sort: Name [asc] "` |
| Remote (right) | `" {user}@{host}:{path} -- Sort: {field} [{dir}] "` | `" root@server:~ -- Sort: Name [asc] "` |
| Remote (connecting) | `" {user}@{host} -- Connecting... "` | `" root@server -- Connecting... "` |
| Remote (error) | `" {user}@{host} -- [red]Error[-] "` | `" root@server -- [#FF6B6B]Error[-] "` |

Title alignment: `tview.AlignLeft` (path should be left-aligned for readability).

## Multi-Select Visual

Source: D-13, UI-04

- Selected files get a `*` prefix before the file name in the Name column
- Selected file rows get `#FFD700` (gold) text color on the file name cell
- Status bar shows selected count when > 0 files selected: `"[#FFD700]{N} files selected[-]"`

## SFTP Connection States (Right Pane)

Source: D-07, D-08, D-09

| State | Right Pane Display | Status Bar |
|-------|-------------------|------------|
| Connecting | Centered text: `"Connecting..."` (SecondaryText) | `"[#A0FFA0]Connecting to {user}@{host}...[-]"` |
| Connected | File listing (normal Table) | `"[#A0FFA0]Connected: {user}@{host}[-]"` |
| Failed | Red error text: `"[#FF6B6B]SFTP connection failed:\n{error}[-]"` | `"[#FF6B6B]Connection failed: {short_error}[-]"` |
| Loading directory | Current listing remains, status bar: `"Loading..."` | `"[white]Loading...[-]"` |

All SFTP operations run in goroutines with `app.QueueUpdateDraw()` for UI updates.
Source: RESEARCH.md Pitfall 2 (blocking prevention), Pattern 3 (NewClientPipe)
